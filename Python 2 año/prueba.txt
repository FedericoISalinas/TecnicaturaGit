Programamos con diferentes tipos de archivos, ahora con txt. 
Los acentos son importantes para las palabras. 
como por ejemplo: accion, ejecucion y produccion. 
La letra w significa write y es para escribir. 
La lerea R significa read y es para leer. 
La letra A significa append y es para cuando queremos anexar mas informacion. 
La letra X sirve para crear un archivo. 
La letra t significa text y es para espesificar el tipo de archivo. 
La letra b sirve para crear un archivo de tipos binarios. 
La letra w+ sirve para crear un escivir pero tambien para leer informacion. 
La letra r+ sirve para crear un abrir, leer informacion. 
Con esto terminamos.
